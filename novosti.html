Novosti o Bootstrap-u

Novosti

Pogledajte koje su inovacije zastupljene u poslednjoj Bootstrap 4 alpha verziji:

Sass

Izvorni kod napisan je u Sass-u. Prethodne verzije koristile su Less, kao glavni predprocesor. Sass je generalno popularniji među front-end zajednicom. Takođe, brže se kompajlira zahvaljujući Libsass-u.

Flex Box Grid

Iako IE 9 ne podržava Flexbox, to nije sprečilo kreatore Bootstrap-a da ga dodaju u verziju 4. U početnim verzijama 4-ke, opcija korišćenja Flex Box-a bila je isključena, a sada je ona podrazumevano uključena. Postoji jedna boolean promenljiva u _variables.scss fajlu sa nazivom $enable-flex. Ako se postavi na false i rekompajlira, sve grid komponente koristiće standardnu Float tehniku.

Unapređeni Medija Upiti

Na prethodnim verzijama često se sticao utisak da fali još jedna prelomna tačka za mobilne uređaje u landscape orijentaciji. Srećom, developeri su to prepoznali i u verziju 4 uključili tu dodatnu prelomnu tačku. Ovako bi izgledali media upiti sa mobile-first pristupom:
→

                  // Extra small devices (portrait phones, less than 576px)
                  // No media query since this is the default in Bootstrap

                  // Small devices (landscape phones, 576px and up)
                  @media (min-width: 576px) { ... }

                  // Medium devices (tablets, 768px and up)
                  @media (min-width: 768px) { ... }

                  // Large devices (desktops, 992px and up)
                  @media (min-width: 992px) { ... }

                  // Extra large devices (large desktops, 1200px and up)
                  @media (min-width: 1200px) { ... }

IE 8 gubi podršku

Sa pojavom Bootstrap 4, podrška za stari IE8 browser je ukinuta. To znači da front-end framework nije ograničen starim pregledačima i sada se potpuno može okrenuti novim CSS3 tehnikama. IE 9 je i dalje podržan, ali najveći problem predstavlja njegovo odsustvo podrške za Flex Box.

Relativne mere

Ovo je jedna od najkonfuznijih promena. Iako je i dalje moguće koristiti px, em, ili pt, tipografija u Bootstrap 4 je kompletno bazirana na rem jedinicama. To znači da su veličine fonta relativne u odnosu na root element (html tag). Podrazumevano, veličina fonta root elementa je 16px.
